# NLP in Python

Course repository for the **Natural Language Processing (NLP) in Python** class in the Master's in Data Science program at HSE University.

The materials move from classical NLP foundations to modern neural and transformer-based methods, with hands-on notebooks in Python and Jupyter.

## Repository Overview

- `codes/`: lab notebooks and in-class coding materials
- `slides/`: lecture slides in `pdf/` and `pptx/`
- `syllabus/`: course syllabus source and exported PDF
- `course presentation/`: course-level presentation assets
- `projects guide/`: project instructions and supporting material
- `textbook/`: textbook-related resources

## Lab Progression

The labs are organized by topic and gradually build up the main ideas used in modern NLP:

- `lab1`: tokens and text preprocessing
- `lab2`: n-gram language modeling
- `lab3`: sparse representations and basic embeddings
- `lab4`: dense embeddings with `gensim`
- `lab5`: attention and multi-head attention from scratch
- `lab6`: getting started with Hugging Face pipelines and models
- `lab7`-`lab8`: reserved for later course topics

## Main Topics

- Tokenization and normalization
- N-gram models
- Classical text representations
- Static word embeddings
- Neural NLP foundations
- Attention and transformers
- Hugging Face workflows for modern NLP

## Working With The Materials

Most materials are provided as Jupyter notebooks and are intended to be run in a local Python environment or in Google Colab.

Typical dependencies used across the course include:

- `python`
- `jupyter`
- `pytorch`
- `transformers`
- `scikit-learn`
- `gensim`
- `nltk`
- `spacy`
